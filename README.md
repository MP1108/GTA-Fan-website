# GTA Fan Website
Fan-made web app for GTA V / GTA VI — manage weapons, vehicles, heists, and characters.

🚀 **Features**
- Browse and manage weapons, vehicles, heists, and characters
- AI chat with Franklin
- Heists editor with TinyMCE

🛠️ **Tech Stack**
- PHP + MySQL + HTML/CSS + JS

📂 **Project Structure**
- `/login.php` — login page  
- `/gta.php` — main hub  
- `/weapons.php` — weapons + DMG bars  
- `/vehicles.php` — special vehicles  
- `/heists.php` — heists with editor  
- `/postacie.php` — characters  
- `/chat.php` — AI chat  

⚙️ **Setup**
1. Copy files to `htdocs/GTA-Fan-website-main/`  
2. Create MySQL database `gta` and import SQL schema  
3. Open in browser: `http://localhost/GTA-Fan-website-main/login.php`
